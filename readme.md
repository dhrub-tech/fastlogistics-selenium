# FastLogistics Selenium Automation Framework

## Overview

This project automates the FastLogistics web application using Selenium WebDriver with Java and TestNG.

## Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven
- Git
- GitHub

## Features

- Login Automation
- Shipment Creation
- Shipment Search
- Page Object Model

## Project Structure

```
src/
pages/
tests/
utils/
base/
```
